## Проекты

### 🪑 Booking Seats

Командный backend-проект на FastAPI для системы бронирования мест в кафе.
В рамках проекта отвечал за разработку функционала, связанного с кафе и столами.

Что реализовано:

* модели кафе и столов
* REST API для управления кафе и столами
* бизнес-логика взаимодействия с сущностями кафе/столов
* работа с базой данных (PostgreSQL + SQLAlchemy)
* интеграция в общий backend проекта на FastAPI

🔗 [Репозиторий](https://github.com/eeXaiLee/booking_seats)
